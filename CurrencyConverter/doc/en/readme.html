Overview

The Currency Converter NVDA Add-on is a professional tool designed to enhance the accessibility of currency conversion for users of the NonVisual Desktop Access (NVDA) screen reader. Developed by Sujan Rai, this add-on provides seamless conversion between multiple currencies with an intuitive interface optimized for accessibility.

Key Features

  • Supports conversion between a wide range of global currencies.
  • Accessible dialog-based interface for selecting currencies and inputting amounts.
  • Configurable default currency pair (e.g., USD to NPR).
  • Integration with NVDA’s tools menu and customizable keyboard shortcut (default: NVDA+Alt+C).
  • Error handling for invalid inputs or network issues during conversion.
  • Promotional dialog to connect users with the developer’s Telegram channel and blog.

Installation

To install the Currency Converter NVDA Add-on:

  1. Download the latest stable version (0.2) from the official release page.
  2. Open NVDA and navigate to NVDA Menu > Tools > Manage Add-ons.
  3. Select Install and choose the downloaded .nvda-addon file.
  4. Restart NVDA to activate the add-on.

Usage

To use the Currency Converter:

  1. Press NVDA+Alt+C or select Currency Converter from the NVDA Tools menu.
  2. In the welcome dialog, choose to join the Telegram channel, read the blog, or proceed by selecting No Thanks.
  3. In the conversion dialog, select the source and target currencies from dropdown menus.
  4. Enter the amount to convert using the spin control.
  5. Click Convert to view the result or Exit to close the dialog.
  6. Results are displayed in a read-only text area with options to go back or exit.

Configuration

The add-on allows customization of default currency pairs:

  • Default source currency: USD
  • Default target currency: NPR
  • Modify these settings via the add-on’s configuration file or through the conversion dialog.

Keyboard shortcuts can be customized via NVDA Menu > Preferences > Input Gestures > Currency Converter.

Keyboard Shortcuts

The default shortcut to open the Currency Converter is:

  • Open Currency Converter: NVDA+Alt+C

Additional shortcuts within dialogs:

  • Escape: Close the current dialog.
  • Enter: Activate the focused button (e.g., Convert, Exit).

Compatibility

The Currency Converter NVDA Add-on is compatible with:

  • NVDA version 2023.2 and later.
  • Windows operating systems supported by NVDA.

Ensure you have an active internet connection for real-time currency conversion data.
